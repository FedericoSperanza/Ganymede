# Ganymede
Ganymede App, that is connected with Themisto

Run this command shell/bash:
<br/>
npm run start:dev
<br/>

Default port 3000 on LocalHost <br/>
EndPoints <br/>
https://ganymede-api-fsperanza.herokuapp.com/
